Changelog:
1.  Enable more Units constructors to build units on the fly
    -   Units(Quantity, Symbol), Units(Quantity, AbstractSring), Units(Pair{<:Union{Symbol,AbstractString}, <:Union{Quantity,Units}})
    -   Units(AbstractUnitLike, AffineTransform, Symbol)
        -   Currently this asserts if the unit is a dimensional alias (like u"m", but things like u"km" won't work)
        -   We can generalize this by renaming the "todims" argument as "tounits" and using 
            Units(dimension(u), tobase(u) ∘ tounits, symbol) #Throw error if (tobase(u) ∘ tounits) isn't an AbstractTransform
2.  AffineTransform now only uses matmul for numeric values, uses normal operators for other cases
3.  Functions with unitless output (exp, sin, cos etc) now produce Real/Complex instead of Quantity{T, NoDims} (matching Unitful behaviour)
4.  Loosened rules around LogQuant operations

Current Patch:
#Hypthetical structure (seems redundant because we could jsut have Quantity{LogQuant{T,LU}, U})
struct LogRate{T, UL, UQ}
    value :: LogQuant{T, UL}
    units :: UQ
end
LogRate *|/ Number = LogRate
LogRate + LogRate = LogRate #fails if UQ are different 
LogRate + LogQuant = LogQuant #fails if UQ is not unitless
LogQuant *|/ Quantity = LogRate 
LogQuant *|/ Units = LogRate #Fails if units are non-scalar 
exp(LogRate) = Quantity #fails if UQ is not unitless

# Multiplies r.value by uscale(r.units) and then calls logubase(r.value)
logubase(r::LogRate) = LogRate(uscale(r.units)*logubase(r.value), dimension(r.units))

import DynamicQuantities
import Unitful
using BenchmarkTools

v1flex = ubase.([1u"m/s", 1u"J/kg", 1u"A/V"])
v1uni  = [1*Unitful.u"m/s", 1*Unitful.u"J/kg", 1*Unitful.u"A/V"]
v1dyn  = [1*DynamicQuantities.u"m/s", 1*DynamicQuantities.u"J/kg", 1*DynamicQuantities.u"A/V"]

@btime sum(x->x^0.0, v1uni)
@btime sum(x->x^0.0, v1flex)
@btime sum(x->x^0.0, v1dyn)

t1flex = ubase.((1u"m/s", 1u"J/kg", 1u"A/V"))
t1uni  = (1*Unitful.u"m/s", 1*Unitful.u"J/kg", 1*Unitful.u"A/V")
t1dyn  = (1*DynamicQuantities.u"m/s", 1*DynamicQuantities.u"J/kg", 1*DynamicQuantities.u"A/V")

@btime sum(x->x^0, t1uni)
@btime sum(x->x^0, t1flex)
@btime sum(x->x^0, t1dyn)
